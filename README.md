# Génération des pages pour la conformité RGPD

##Description du plugin WP-RGPD
Wp-RGPD vous permet de générer 4 pages necessaires pour suivre l'ogligation RGPD :
*CGU
*CGV
*Mentions légales
*Politique de confidentialité
